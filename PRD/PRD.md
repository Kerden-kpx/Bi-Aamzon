# 基于BSR类目数据的 BI 分析与决策系统 - PRD

## 1. 项目概述
### 1.1 项目目标
搭建一套面向亚马逊类目（以 Reciprocating Saw Blades 为试点）的可交互 BI 分析与决策系统。支持 BSR Top 100 可视化、拖拽式目标坑位设定、竞品对比及运营策略闭环管理。

### 1.2 预期收益
- 提升类目整体布局效率，从单链接决策升级为矩阵决策。
- 明确自家品牌（EZARC/TOLESA）在不同排名段的竞争位置与差距。
- 沉淀运营策略，转化为可管理的 To Do 待办。

## 2. 核心功能模块与任务梳理

### 2.1 类目 BSR 榜单模块 (左侧)
- [ ] **BSR 前 100 列表展示**：支持卡片/列表视图展示 Top 100 链接及其核心指标（主图、标题、价格、BSR、星级等）。
- [ ] **悬停详情 (Hover)**：显示变体数、大类排名、估算销量、流量占比等深度数据。
- [ ] **筛选与搜索**：支持按品牌、价格区间、星级等基础维度筛选。
- [ ] **自定义标签系统**：支持对 BSR 链接打标签（如“重点监控”、“低价竞品”），并在卡片展示。

### 2.2 自家产品库模块 (右侧)
- [ ] **内部产品线展示**：基于 SKU/ASIN 维护表，展示 EZARC/TOLESA 的在售产品。
- [ ] **应用场景筛选**：支持按应用标签（切木/切枝等）、材料、定位等标签筛选。
- [ ] **关键词搜索**：支持通过 SKU、ASIN 或简称快速定位产品。

### 2.3 交互式决策模块 (核心)
- [ ] **拖拽匹配功能**：支持将竞品 ASIN 拖拽至自家产品，或反向拖拽，建立“目标坑位”映射。
- [ ] **竞品/自家对比 (底部)**：自动拉取两者的核心指标（价格、销量、星级、流量占比等）进行同屏对比并标示差值。
- [ ] **策略备忘与 To Do**：集成文本输入区域，支持保存策略并自动生成关联的 To Do 任务。

### 2.4 数据透视分析模块
- [ ] **品牌份额统计**：可视化展示 Top 100 中各品牌的坑位占比、销量占比。
- [ ] **趋势对比分析**：保留 BSR 周维度/双周维度快照，支持历史趋势查看。

### 2.5 任务管理与权限控制
- [ ] **To Do 待办面板**：提供所有策略任务的列表视图，支持状态更新、负责人分配、优先级筛选及导出。
- [ ] **权限角色**：区分管理员（全权限）与运营人员（个人任务管理）。

## 3. 数据需求
### 3.1 外部爬取字段 (BSR Top 100)
- ASIN, Parent ASIN, Title, URL, 图片, 品牌, 价格, 星级/数, BSR 排名, 大类排名, 变体数, 上架时间, 估算销量/销售额, 转化率估算, 流量占比估算。

### 3.2 内部维护字段
- SKU, ASIN, 品牌, 应用/齿形/材质标签, 定位标签, 在售状态。

## 4. 验收标准
- [ ] **功能闭环**：能够从查看到对比，再到策略生成，流程无死角。
- [ ] **数据准确性**：核心 BSR 排名与价格字段需与亚马逊前台保持一致。
- [ ] **交互流畅度**：拖拽操作响应灵敏，对比窗口切换自然。
